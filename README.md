# Aviva-style Insurance Portal (Demo)

Simple demo app with authentication and a protected dashboard.

## Requirements
- Node.js 16+ recommended

## Run locally

1. Install dependencies

```bash
npm install
```

2. Start server

```bash
npm start
```

3. Open http://localhost:3000 in your browser

Notes: This is a demo scaffold (SQLite, simple session store). Do not use in production without securing secrets, HTTPS, CSRF protection and proper validation.
